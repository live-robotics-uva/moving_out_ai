Game Controls

Keyboard Controls:

Agent 1:
  • WASD: Move up, left, down, right
  • H (hold): Move backwards
  • Spacebar: Pick up or drop item
Agent 2:
  • Arrow Keys: Move up, left, down, right
  • Ctrl (hold): Move backwards
  • Enter: Pick up or drop item

Joystick Controls:

  • Left Stick: Move the agent
  • R Button: Pick up or drop item
  • ZL (hold): Move backwards

Note: The keyboard controls are only available for testing purposes and cannot control all directions and movement speeds. Using joystick controls is more aligned with the environment design and provides better gameplay experience.
